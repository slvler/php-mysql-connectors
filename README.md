#### Insert Method (TableName, Data = array()):

TableName, Data:

```php
$tableName = "Orders";

$data = Array
(
  "ShipName" => "Blue",
  "ShipAddress" => "Nottingham",
  "ShipCity" => "UK"
);
```

```php
Insert Method
<?php
$methodInsert = $method->Insert($tableName,$data);
?>
```




#### Update Method (TableName, Id = array() , Data = array()):



TableName, Id , Data:

```php
$tableName = "Orders";

$Id = array(
 "OrderID" => "1"
);

$data = Array
(
  "ShipName" => "Blue",
  "ShipAddress" => "Nottingham",
  "ShipCity" => "UK"
);
```

```php
Update Method
<?php
$methodUpdate = $method->Update($tableName', $id, $data);
?>
```
